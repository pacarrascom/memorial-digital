# Dorsera Memorial

Plataforma de memoriales digitales conectados por código QR, parte del ecosistema Dorsera.

## Estructura del repositorio

- `docs-fundacion.md` — visión de producto, arquitectura, modelo de datos, diagrama ER, RBAC, sistema de diseño, roadmap y monetización
- `supabase-migrations/` — migraciones SQL (esquema, RLS, funciones) a aplicar en orden vía Supabase SQL Editor o CLI. Ver `supabase-migrations/README.md`
- `apps-web/` — frontend Next.js. Ver `apps-web/README.md` para integración y variables de entorno

## Siguientes entregables

Panel de administración familiar, autenticación, generación de QR, panel de funeraria (B2B), árbol genealógico interactivo.
